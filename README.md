# India's Agriculture Crop Production Analysis (Tableau)

This repository contains a **Tableau-based data visualization project** that analyzes India's agricultural crop production using historical data from **1997–2021**. The project transforms raw agricultural data into interactive dashboards and stories to provide insights into crop production trends, seasonal variations, state-wise performance, and regional productivity.

## Features

*KPIs – Total Production & Total Yield (2 KPI tiles)
*Trend of Agricultural Yield (1997–2021) – Line Chart
*State-wise Agricultural Production Distribution – India Map
*Crop-wise Total Production Analysis – Horizontal Bar Chart
*State-wise Cultivated Area Comparison – Grouped Bar Chart
*Leading States by Agricultural Production – Bar Chart
*Year-wise Total Agricultural Production (1997–2021) – Bar Chart
*Production Distribution by Unit Type – Donut Chart
*Season-wise Agricultural Production Share – Pie Chart

## Objectives

* Analyze historical agricultural production data.
* Identify high-performing crops and regions.
* Understand seasonal and yearly production patterns.
* Support data-driven decision-making for farmers, policymakers, researchers, and agribusiness investors.
* Demonstrate Tableau's capabilities for agricultural data visualization.

## Dataset

* **Time Period:** 1997–2021
* **Data Includes:** State, District, Crop, Season, Year, Area Cultivated, and Production.

## Tools & Technologies

* Tableau Desktop
* Microsoft Excel / CSV
* Data Cleaning & Visualization

## Project Outcomes
The dashboards help users explore agricultural trends, compare crop production across states and seasons, identify productivity patterns, and gain actionable insights through interactive visualizations.
